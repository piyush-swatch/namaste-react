## Parcel package
    - Dev Build
    - Local server
    - HMR (Hot module replacement) - automatically refreshes the content on webpage using the file watching algorithm (written in C++)
    - Caching - Faster builds - created .parcel-cache files
    - Image optimization
    - Minification
    - Bundling of files
    - Compression of files
    - Consistent hashing
    - Code splitting
    - Differential bundling : to support older browser
    - Diagnostics
    - Tree Shaking - remove unused codes
